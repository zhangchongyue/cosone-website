# COSONE 官方网站

专业单板滑雪装备品牌官网，参考Burton风格设计。

## 项目结构

```
COSONE-Website/
├── index.html          # 首页
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互脚本
├── images/             # 图片文件夹
└── README.md           # 说明文件
```

## 如何预览

1. 将图片放入 `images/` 文件夹
2. 用浏览器打开 `index.html`
3. 或者使用Live Server等本地服务器

## 功能特性

- ✅ 响应式设计（PC/平板/手机）
- ✅ 黑色导航栏（参考Burton）
- ✅ 全屏Hero大图
- ✅ 产品分类网格
- ✅ 精选产品展示
- ✅ 品牌故事横幅
- ✅ 滑手Team展示
- ✅ 邮件订阅
- ✅ 购物车交互
- ✅ 快速查看弹窗
- ✅ 收藏功能

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (无框架依赖)
- Intersection Observer API

## 浏览器支持

- Chrome / Edge / Firefox / Safari 最新版
- IE11 及以上
